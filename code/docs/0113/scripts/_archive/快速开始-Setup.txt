  DreamBook Kiosk 配置 - 快速开始
═══════════════════════════════════════════════

配置 Kiosk 模式（自动登录并运行 DreamBook）：

  1. 双击 run-setup-as-admin.bat
  
  2. 点击"是"（UAC提示）
  
  3. 等待配置完成（约1分钟）
  
  4. 部署应用到 C:\kiosk\dreambook\dreambook.exe
  
  5. 重启电脑

═══════════════════════════════════════════════

配置内容：
  ✓ 创建 kiosk 用户 (密码: DreamBook2026!)
  ✓ 开机自动登录
  ✓ 启动 DreamBook 替代桌面
  ✓ 启用触摸键盘

═══════════════════════════════════════════════

如果上面的方法不行：

  方案A：
  - 右键 setup-kiosk-en.bat
  - 选择"以管理员身份运行"
  
  方案B：
  - 以管理员身份打开 PowerShell
  - 运行: .\setup-kiosk.ps1

清理配置（恢复正常模式）：

  双击 run-cleanup-as-admin.bat
